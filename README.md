# Article Management API

A RESTful API built using ASP.NET Core implementing Clean Architecture principles.

## Tech Stack

- ASP.NET Core Web API
- Entity Framework Core
- JWT Authentication
- Swagger API Documentation
- Clean Architecture

## Features

- User Registration
- Login Authentication
- Create Articles
- Favorite Articles
- Secure API using JWT
- Swagger API Testing

## Project Structure

src/
  Conduit/
    Domain
    Features
    Infrastructure

tests/
  Conduit.IntegrationTests

## Run the Project

dotnet restore
dotnet build
dotnet run
